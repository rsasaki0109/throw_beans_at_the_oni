## ゲームルール

### 勝利条件 / 終了条件

- **終了条件**: 10秒経過
- **勝利条件**: なし（スコアアタック形式）
- **目標**: 自己ベスト更新 / リーダーボード上位

### 操作方法

| 操作 | PC | モバイル |
|------|-----|---------|
| 照準 | マウス移動 | 画面タッチ位置 |
| 投げる | 左クリック / スペース | タップ |
| 連射 | 長押しで連続投擲 | 長押しで連続投擲 |

### スコア仕様

| ヒット対象 | 得点 |
|-----------|------|
| 通常鬼（青） | 10点 |
| 高速鬼（赤） | 30点 |
| ボーナス鬼（金） | 100点（低確率出現） |
| コンボボーナス | 連続ヒットで×1.5倍 |

### リトライ導線

- 結果画面に「もう一回」ボタンを大きく配置
- 3秒放置で自動リスタート
- リザルト画面でスコア共有ボタン表示

---

## ゲームフロー

```
[ロビー/待機]
    ↓ スタートボタン押下
[カウントダウン] 3...2...1...GO!（2秒）
    ↓
[プレイ中] 10秒間の豆投げタイム
    ↓ タイマー0
[結果表示] スコア・ランキング表示（3秒）
    ↓ ボタン or 自動
[リプレイ] → プレイ中へ戻る
```

### 各フェーズ詳細

| フェーズ | 時間 | 内容 |
|---------|------|------|
| カウントダウン | 2秒 | 画面中央に数字表示、鬼が出現開始 |
| プレイ中 | 10秒 | 鬼が画面内を移動、プレイヤーが豆を投擲 |
| 結果表示 | 3秒 | スコア・ハイスコア・順位を表示 |

## オブジェクト構成

### プレイヤー

- 画面手前に固定（一人称視点）
- 見た目のアバターは非表示（豆を持つ手のみ表示）
- 投擲アニメーション：腕を振る動作

### 鬼

| 種類 | 色 | 動き | 出現率 |
|------|-----|------|--------|
| 通常鬼 | 青 | ゆっくり左右移動 | 70% |
| 高速鬼 | 赤 | 素早くランダム移動 | 25% |
| 金鬼 | 金 | 一瞬だけ出現して消える | 5% |

- サイズ：人型、プレイヤーの1.5倍程度
- ヒット判定：頭・胴体に当たり判定
- 被弾時：後ろに吹き飛んで消滅

### 豆

- 形状：球体（やや楕円）
- 色：薄茶色
- 軌道：放物線（重力あり）
- 発射レート：0.15秒間隔（連射時）

### UI

| 要素 | 位置 | 内容 |
|------|------|------|
| タイマー | 画面上部中央 | 残り秒数（大きく表示） |
| スコア | 画面右上 | 現在スコア |
| コンボ | スコア下 | 連続ヒット数（×1.5表示） |
| 照準 | 画面中央 | シンプルな十字線 |

## 演出・体験設計

### 気持ちよさのポイント

1. **即時フィードバック**: 当たった瞬間に効果音＋スコアポップアップ
2. **連射の快感**: 長押しで豆が連続で飛ぶ爽快感
3. **コンボ演出**: 連続ヒットで画面が揺れ、色が変わる

### ヒット時のフィードバック

| 要素 | 演出 |
|------|------|
| 視覚 | 鬼が吹き飛ぶ + 星エフェクト + スコアポップ |
| 聴覚 | 「パコン！」という軽快なSE |
| 触覚 | モバイルはバイブレーション（短） |

### 短時間でも中毒性を出す工夫

- **スコア差の明確化**: 100点単位で大きく変動
- **ランダム性**: 金鬼出現で「もう一回」を誘発
- **即リトライ**: 結果画面からワンタップで再開
- **ハイスコア表示**: 自己ベスト超えると派手な演出

## MVP定義

### 「これだけあれば公開する」最小構成

| カテゴリ | 必須要素 |
|----------|---------|
| ゲームプレイ | 10秒タイマー + 豆投擲 + 鬼1種類 |
| UI | タイマー + スコア表示 + リトライボタン |
| 演出 | ヒットSE + 鬼消滅エフェクト |
| 操作 | PC：クリック / モバイル：タップ |

### MVP除外（v2以降）

- 高速鬼・金鬼
- コンボシステム
- リーダーボード
- スキン・課金要素
- BGM

### MVP完了基準

1. ゲーム開始→プレイ→結果→リトライの一連フローが動作
2. PC・モバイル両方で操作可能
3. スコアが正しく加算される
4. 致命的なバグがない
